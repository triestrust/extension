Privacy Policy

Effective Date: May 26, 2026

Overview

Trust Trigon respects user privacy and is designed to operate with minimal permissions while providing browser-based SEO tools directly inside the browser. The extension does not collect, sell, store, or share personal information.

Information Collection

Trust Trigon does not collect, monitor, or transmit any sensitive or personal user data.

  • No personally identifiable information is collected.
  • No passwords, payment data, or account credentials are accessed.
  • No browsing history or browsing activity is tracked.
  • No website content, form data, or keystrokes are monitored.
  • No analytics trackers, advertising identifiers, or fingerprinting technologies are used.

Permissions & Usage

The extension may request limited browser permissions strictly required for functionality.

  • storage — Used only to save local extension settings inside the user's browser.
  • clipboardRead / clipboardWrite — Required only for copy and paste functionality initiated by the user.
  • https://firestore.googleapis.com/* — Used only to retrieve a small read-only Firebase configuration document from our own project for extension behavior control.

Remote Configuration

Trust Trigon may use a secure Firebase-hosted read-only configuration file to manage extension behavior, such as displaying built-in tools or loading approved webpages. No remote code execution is used, and no personal information is collected, analyzed, or transmitted through this process.

Third-Party Services

The extension may occasionally use trusted external services required for features such as QR code generation or other optional utility functionality. These services are used only when requested by the user.

Data Security

All SEO tools operate locally inside the browser environment. Trust Trigon does not maintain external servers for storing user-generated content, personal information, browsing data, or extension activity.

Policy Updates

This Privacy Policy may be updated periodically to reflect extension improvements, feature updates, or Chrome Web Store policy requirements.
